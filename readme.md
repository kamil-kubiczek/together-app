### About project

This project is example of fullstack web app based on BaSS - Supabase.
App allows to connect multiple people in form of posts available to everyone.

#### Quick features summary:

-  Register
-  Login
-  Logout
-  Remove account
-  CRUD posts

#### Quick tools summary:

-  Nuxt 3
-  Supabase
-  Eslint
-  Prisma
-  Prettier
-  Vitest
-  Figma
